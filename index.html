Elevator Pitch

Problem Description

Outdoor recreation areas such as parks, rivers, lakes, and beaches accumulate large amounts of trash. Many independent organizations and local governments spend time and resources trawling these areas looking for litter, debris, and dumping to clean up. Not everyone that visits these locations are capable or ready to pick up large amounts of trash. Some large litter and dumping requires planning and/or equipment for proper and safe removal. Many of these areas are also very large and clean up crews must walk great distances in search of any trash that must be picked up.

Customer and End User

Primary Users

END USERS: Community members such as hikers and beachgoers who encounter trash while participating in an outdoor activity and want to report it for cleanup.

Secondary Users

CUSTOMERS: Cleanup organizations such as park services and waste management departments that plan for trash removal.

Solution Statement

Provide the public a way of quickly communicating the scope and location of found litter, trash, and dumping to organizations willing and interested in clean up. A system where users can easily upload geotagged photos of litter or dumping, thus providing a map of areas to clean for the city, volunteer organizations, or even motivated individuals so the trash could be removed far more quickly and efficiently. The photos serve as easy visual identification of the trash as well as the scope of the problem. Over time, the data can even be used for public education or even city officials to dispatch people to problem areas as a preventative measure.

Project Description

The Problem and our Solution

Problem Statement

Independent organizations and local governments struggle to efficiently locate and remove litter in outdoor recreational areas such as parks, rivers, lakes, and beaches. Without knowing locations and characteristics of waste items, volunteer efforts can be left unprepared and ineffective. Communities lack a live reporting system to connect citizens with organizations capable of safe and efficient waste removal efforts.

Problem Characteristics

Resources

  • Organizations struggle to track illegal dumping.
  • Items too much for individuals.
  • Users and Litterers alike may or may not have adequate knowledge of proper disposal options.

Communications

  • Users don’t know who to contact regarding trash.
  • Users lack tools focused on organizing cleanup efforts.
  • User reports come in one at a time and require coordination within cleanup organizations to increase efficiency.

Convenience

  • Users can’t communicate in the moment when they see the trash.
  • They’ve forgotten by the time they return home.
  • Litterers often dump out of convenience on private or public property

Complexity

  • Users reach out to orgs via email/social media which may get lost or go unnoticed.
  • Thus, users contact multiple orgs at a time to ensure attention.

Solution Statement

TrashTag aims to improve the litter and waste cleanup process by developing a mobile application where users can photograph and report piles of waste, give their exact locations, and connect with cleanup groups that are well-equipped to handle the cleanup process in order to improve environmental wellbeing and preserve nature’s beauty for the public eye. We are committed to helping the environment and the public by providing a resource that clearly highlights the location, extent, type and weight of the waste as well as provides updates regarding existing cleanup efforts, dangers and geographic littering patterns to help cleanup crews allocate their time and resources more effectively.

Solution Characteristics

Live Reporting and Mapping

  • Users snap a picture of the litter and geotag it in the app for it to be uploaded to an interactive map.
  • This allows organizations to properly allocate time, resources, staff, and funding.

Litter Report System

  • The images provide users with a visual understanding of the problem.
  • This allows them to prepare resources for proper clean up.

"Litter Hero!"

  • A reward system for users who successfully clear litter locations from the map.
  • This allows users track their progress and notice measurable impact.

Close the Loop

  • When users are near reported trash they receive a request to verify the trash is still there or if its been previously cleared.

User Stories

How TrashTag functions in our community

CITIZEN USER STORIES

As a Citizen,

  • I want to report the location of where trash is located so that other users can see where the issue is at and potentially clean it up.
  • I want to upload photos of litter I find so that others can see the issue, clean it, and mark it as cleaned.
  • I want to be able to report the litter type so that others are able to know if it can be safely disposed of or if assistance is required.
  • I want to mark the severity level of reported trash so that others can know how much litter is in the reported area.
  • I want to request cleanup verification status of the affected area so that it can be verified if it still exists or not.
  • I want to be able to interact with a map that locates litter-affected areas so that I can see where the litter is at.
  • I want to view the map with a “litter density” filter so that I can know which areas are heavily affected.
  • I want to be able to filter the map by “cleanup status”, “litter type”, and “location” so that the map can be easily accessible to users.
  • I want to coordinate, join, and view local cleanup events so that I can help with cleaning up litter.
  • I want to claim cleanup areas during cleanup events so that I can help with cleaning up litter.
  • I want to delegate cleanup tasks to other users when coordinating events so that cleanup events will run smoothly.
  • I want to be able to RSVP and check-in to local cleanup events so that my participation in the event(s) can be logged in the app.
  • I want to be able to see and click on app notifications about new litter-affected areas that get reported by other users so that I can be alerted to verify it.
  • I want to be able to receive proximity cleanup alerts and notifications so that I can be alerted to such events and plan accordingly.
  • I want to be able to update cleanup status so that cleanup tracking can be manually updated by users.
  • I want to upload proof of the cleanup, such as images, so that it can be accurately verified by other users.
  • I want to be able to monitor cleanup status so that I can know what needs to be cleaned up and where.
  • I want to verify accurate cleanup status so that repeated reports do not occur in report listings.
  • I want to flag suspicious content so that only litter-affected areas are getting properly reported.
  • I want to view impact statistics so that I can be motivated to help clean my community.
  • I want the app to have a reward system to that I can stay motivated to participate in clean up events.
  • I want to see a ranked list of “Top Contributors” so that I can easily track my reward progress against other users.
  • I want to be able to access the Community Guidelines so that there is clear communication between users.
  • I want my personal information to remain private so that I can safely report issues without fear of exposure.
  • I want to be able to mark certain reports as spam so that I can know if I need to flag a moderator to review it.

As a Local Government,

  • I want to view aggregated litter trends and density maps so that I can identify problem areas.
  • I want to access statistics about litter reports and cleanups so that I can plan budgets and policies.
  • I want citizen data to remain anonymous so that privacy laws and ethics are upheld.
  • I want to receive periodic reports about cleanup progress so that I can track the impact on municipal costs.
  • I want to embed the TrashTag map on city websites so that citizens can access the data easily.
  • I want to be able to easily put the TrashTag map on city websites so that more people be informed about cleanups.
  • I want to be able to coordinate city wide clean up events so that they can use direct government resources.
  • I want to be able to claim cleanup areas so that hired volunteers can clean them directly.
  • I want to be able to delegate hired volunteers to a cleanup area so that we can communicate on the cleanup site without needing to be face to face.
  • I want to view upcoming cleanups so that we can advertise them on our website.
  • I want to RSVP to local events so that other volunteers know our volunteers will be there.
  • I want to check into events so that other volunteers know our volunteers are there.

MODERATOR USER STORIES

As a Moderator,

  • I want to be able to monitor and review user-submitted reports so that I can automatically remove inappropriate or duplicate data.
  • I want to respond to post reports from users so that the platform remains trustworthy.
  • I want tools to correct or delete false or spam submissions so that data analytics remain accurate.
  • I want to be able to create and enforce clear community guidelines so that the app can foster a healthy, engaging, and collaborative online community for users.
  • I want to be able to delegate tasks so that a misassigned task can be corrected.
  • I want to be able to view cleanup events so that I can monitor that event.

Glossary

Key Terms and their respective defintions

Illegal Dumping

The unauthorized disposal of trash and other hazardous materials onto private and or public property.

Metadata

Descriptive information within data files such as date, location, and timestamp data in photos to be submitted to Trashtag.

Geotagging

The addition of GPS coordinates to images that enable precise location identification where trash was reported.

Live Real-Time Updates

Live data synchronization displaying trash reports to both users and cleanup organizations instantaneously.

Gamification

The incorporation of game-like mechanics such as rankings to motivate user participation and skyrocket cleanup operations.
